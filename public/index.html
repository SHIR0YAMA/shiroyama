Por que vincular a conta?

Para enviar arquivos diretamente para você, nosso site precisa saber qual é a sua conta no Telegram.

Ao vincular, você permite que nosso bot responda às suas solicitações. Quando você clica em "Receber", o site diz ao bot: "Envie este arquivo para o usuário que tem esta conta vinculada".

É um processo seguro e a conexão só é usada para entregar os arquivos que você solicita.
